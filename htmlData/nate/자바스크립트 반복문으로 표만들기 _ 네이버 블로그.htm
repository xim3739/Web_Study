자바스크립트 for문은 다른 언어(C, Java)와 크게 다르지 않아요. 그러나 HTML 코드들을 같이 끼워 ...

자바스크립트 반복문을 이용해서 표를 만들어볼 계획이랍니다. 이전 시간에 배웠던 것에 비해 조금 응용이 되었다고 생각될 수 있습니다. 하지만, 천천히 생각을 해보면 헷갈리지 않아요! 우선은 이런 식으로 작동하는 것을 구현하려고 합니다.

사용자로부터 줄, 칸 수를 입력 받으면, 그에 맞는 표가 완성. 예를 들면, 사용자가 줄 수에 3, 그리고 칸 수에 2를 입력했다고 가정을 한다면, 아래와 같은 표가 나와야하겠죠? 덤으로 숫자까지 나오게 하는 방식으로 구현을 해볼 예정입니다. 결과는 아래처럼..!!

다른 사용자는 다섯 줄, 세 칸이었다고 가정을 하면, 그에 알맞은 표가 등장하게 하는 겁니다. 어떤 느낌인지 대충 이해를 하셨는지요?

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>자바스크립트 연습</title> <style type="text/css"> table, tr, td{ border : 1px solid black; text-align: center; } td{ width: 20px; } </style> <script type="text/javascript"> var x=Number(prompt("몇 줄?")); var y=Number(prompt("몇 칸?")); var num=0; document.write("<table>"); for(i=0;i<x;i++) { document.write("<tr>"); for(j=0;j<y;j++) { document.write("<td>"+num+"</td>"); num++; } document.write("</tr>"); } document.write("</table>"); </script> </head> <body> </body> </html>


생각보다 그렇게 어렵지는 않습니다. CSS를 활용해, 선 굵기를 조절해주었습니다. 그냥 할 경우 숫자만 나오고 테두리는 없는 것으로 나오므로, 좀 더 확연하게 알기 위해서 위와 같이 꾸며주는 것을 넣어보았습니다.


입력한 결과 원하는대로 잘 나온 것을 확인할 수 있습니다. 이런 식으로 사용자가 원하는 만큼 구현을 할 수도 있다는 것을 알려드리려고 이런 식의 포스트를 구성해보았습니다.
